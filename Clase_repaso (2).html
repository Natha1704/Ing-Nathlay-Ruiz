3. Descripcin grfica de una variable

Introduccin a los grficos en R:

Los comandos de R son funciones que incluyen un conjunto de argumentos separados por comas y todos ellos entre parntesis:

boxplot(datos$peso, main = "primer grfico", xlab = "peso")

hist(datos$peso, main = "primer grfico", xlab = "peso")

Argumentos generales que utilizaremos en los grficos: main: Ttulo principal del grfico

xlab: Etiqueta para el eje X

ylab: Etiqueta para el eje Y

col: vector de colores

3.1. Variables cuantitativas:

Hay varias opciones: box-plots, histogramas, Con un interrogante delante del nombre de la funcin obtenemos ayuda sobre su uso: ?boxplot

Es posible incluir o modificar elementos en el grfico: Incluimos color en la caja

boxplot(edad, col = "red")

Aadimos un ttulo principal y subttulo para el eje X

 boxplot(edad, col = "red", main = "Evaluacin de riesgos laborales", xlab = "Edad")

Modificamos la escala del eje Y

 boxplot(edad, col = "red", main = "Evaluacin de riesgos laborales", xlab = "Edad",
        ylimit = c(0,45))

3.2. Variables categricas:

Comandos en R: table(x) para obtener la tabla de frecuencias pie(table(x)) genera un grfico de sectores barplot(table(x)) genera un diagrama de barras

Los diagramas de barras son preferibles a los diagramas de sectores.

3.2.1 Grficos circulares o de sectores: La variable contrato indica el tipo de contrato de cada trabajador: 1, 2, 3, 4, 5, 6

Preparamos una tabla de frecuencias

Generamos el diagrama de sectores

Seleccionamos colores que nos gusten ms para cada sector del grfico:

pie(table(contrato), col = as.factor(color) )

Adems incluimos un ttulo

pie(table(contrato),  col = as.factor(color), main = "Tipos de contratos")
